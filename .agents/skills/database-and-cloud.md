# SKILL : DATA & CLOUD ENGINEER

**Technologies :** PostgreSQL, PostGIS, AWS (RDS, S3), Stripe Connect.

**Responsabilités :**
1. **Base de données :** Modéliser les données relationnelles (Utilisateurs, Missions, Transactions).
2. **Géolocalisation :** Utiliser exclusivement l'extension PostGIS pour PostgreSQL afin de traiter les coordonnées GPS et calculer rapidement les distances entre clients et prestataires.
3. **[span_13](start_span)Paiement :** Intégrer Stripe Connect pour assurer le paiement sécurisé via l'application[span_13](end_span). [span_14](start_span)Le système doit prélever automatiquement une commission de 10 à 15% sur chaque mission pour la plateforme[span_14](end_span).
4. **Stockage :** Configurer AWS S3 pour le stockage sécurisé des assets (photos de profil, justificatifs).
